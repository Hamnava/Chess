# ♟️ 3D Chess Game with Sound

A modern take on the classic game of chess — rendered in **3D graphics** with immersive **sound effects**. Built to be both visually engaging and technically robust, this project brings chess to life in a whole new way.

---

## ✨ Features
- **3D Board & Pieces**: Fully rendered chessboard and pieces with smooth animations.
- **Sound Effects**: Realistic move sounds, captures, and check/checkmate alerts.
- **Cross-Platform**: Runs on Windows, macOS, and Linux.
- **Game Modes**:
  - Player vs Player (local)
  - Player vs AI (configurable difficulty)
